Building an 8-Bit Computer

I built a fully functional 8-bit computer from scratch using 74LS series logic chips. This included creating a custom clock, ALU, registers, RAM, and control logic wired together on breadboards.

Microcode drives the system, controlling instruction flow through a programmable ROM. I designed a simple instruction set with support for arithmetic, memory access, conditional jumps, and immediate values. Programs are loaded directly into RAM using DIP switches and a manual clock stepper.

Debugging this machine taught me a lot about signal timing, decoupling capacitors, tri-state logic, and the patience needed to trace a broken NAND gate at 2am.

There’s something deeply satisfying about watching machine code execute visibly, LED by LED. This project grounded my understanding of how high-level code eventually becomes logic gate toggles—and gave me a new appreciation for the complexity inside even the simplest CPU.
