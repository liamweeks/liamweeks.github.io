Finding Tumors With Machine Learning

In this project, I trained a machine learning model to detect potential tumors from digitized mammogram images. The dataset consisted of thousands of labeled samples from the Wisconsin Diagnostic Breast Cancer dataset, which includes features extracted from scans like texture, area, and smoothness.

I used a combination of preprocessing, feature selection, and a random forest classifier to build a pipeline that achieved over 95% accuracy on a hold-out test set. Feature importance analysis showed that metrics like radius and concavity played key roles in the model’s decision-making.

Beyond the numbers, this project taught me how to properly evaluate medical ML systems—especially the importance of precision and recall when false negatives can have serious consequences.

In the future, I’d like to extend this work to image-based detection using convolutional neural networks and experiment with attention-based architectures for interpretability.
